# ImmunoQuest GitHub Pages Project

## Structure
- `index.html`
- `style.css`
- `script.js`
- `data/questions.json`
- `manifest.json`

## Déploiement GitHub Pages
1. Dézippe ce fichier.
2. Upload **tout le contenu du dossier** à la racine du dépôt GitHub.
3. Le dépôt doit contenir directement :
   - `index.html`
   - `style.css`
   - `script.js`
   - `data/`
   - `manifest.json`
4. Va dans Settings > Pages.
5. Source : Deploy from a branch.
6. Branch : main, folder : `/ (root)`.
7. Attends 1 à 5 minutes.
